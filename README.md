## Installation

1. Create /usr/local/bin/qunit-test-hash with content from gist.
2. Run `chmod +x /usr/local/bin/qunit-test-hash`.
3. Add `Plug 'plicjo/ember-testing-chrome.vim'` to ~/.vimrc.
4. Run `:PlugInstall`.
